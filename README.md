## Run Locally

Clone the project

```bash
  git clone https://AmirulIslamRafi/My-TailwindCSS-PalyGround.git
```

Go to the project directory

```bash
  cd My-TailwindCSS-PalyGround
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
